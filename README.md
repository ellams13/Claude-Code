# Claude-code

A personal workspace for experimenting with Claude Code — Anthropic's AI-powered CLI for software engineering tasks.

## Overview

This repo serves as a sandbox for exploring what Claude Code can do, from editing files and writing code to managing git workflows and automating day-to-day development tasks.

## Features

- AI-assisted code editing and refactoring
- Automated git commits and GitHub pushes
- File creation and project scaffolding
- Natural language driven development workflow

## Getting Started

Clone the repo and explore the files. No dependencies required.

```bash
git clone https://github.com/ellams13/Claude-Code.git
cd Claude-Code
```
